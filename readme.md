## Description
Pipex is about replicating the behavior of this command line:
<p>< file1 cmd1 | cmd2 > file2<p>
<p>Where cmd1 and cmd2 are commands such as cat or wc -l.</p>

## Instructions
<p>To compile the program, type "make" in the terminal.</p>
<p>To execute the program, type ./pipex file1 cmd1 cmd2 file2</p>
<p>file1 and file2 aren't provided but a simple one line text file should work for file1. For exemple a file called src.txt with a "Hello" will suffice.</p>
<p>file2 doesn't need to be created. But you must at least type the name in the terminal. The program will create file2"</p>
